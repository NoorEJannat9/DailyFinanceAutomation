# Daily Finance Automation

Automated end-to-end test suite for [Daily Finance](https://dailyfinance.roadtocareer.net/) using Selenium + TestNG + Java.

## Tech Stack
- Java 21
- Selenium 4.18.1
- TestNG 7.9.0
- WebDriverManager 5.7.0
- ExtentReports 5.1.1
